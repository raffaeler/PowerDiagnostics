# Diagnostic UI

## Intro

* `npm start`: start the development server (require CORS on the ASP.NET server)
* `npm test`: test the js code
* `npm run build`: publish the app into DiagnosticServer/wwwwroot
* `npm run eject`: don't run this, it's one-way

## Packages

* react-json-tree
* @microsoft/signalr
* react-bootstrap
* react-router-dom@6
